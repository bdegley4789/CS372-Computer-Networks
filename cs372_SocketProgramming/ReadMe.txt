To Compile chatclie.c
Type gcc chatclie.c -o chatlie
on sftp://access.engr.oregonstate.edu server
ignore any warnings

Run chatserve.py
with python chatserve.py
ignore any warnings
For HOST name enter: localhost
For PORT enter: 9999
enter whatever name for the handle you want

Now in another window run chatlie executable
./chatlie
For HOST name enter: localhost
For PORT enter: 9999
enter whatever name for the handle you want
Type a message in to start connection
Now alternate between client and then server sending messages
